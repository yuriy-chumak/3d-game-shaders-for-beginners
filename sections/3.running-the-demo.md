# 3D Game Shaders For Beginners

## Running The Demo

Just run a selected sample file from '3d-game-shaders-for-beginners" sample folder.

For example,
```bash
$ ./5.glsl.lisp

Checking GLX_ARB_get_proc_address support... ok.
Checking GLX_ARB_create_context support... ok.

```
